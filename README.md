**Status:** ✅ Finalizado

Esta é uma **calculadora simples e funcional** desenvolvida com tecnologias web puras: **HTML**, **CSS** e **JavaScript**. O projeto foi construído com o objetivo de treinar conceitos básicos de manipulação de DOM, estilos personalizados e interações com eventos em tempo real.

---

## 🔧 Tecnologias Utilizadas

- **HTML5** – Estrutura do layout da calculadora
- **CSS3** – Estilização com responsividade e visual limpo
- **JavaScript** – Lógica de operações e manipulação dinâmica

---

## 📌 Funcionalidades

✔️ **Operações básicas** – Soma, subtração, multiplicação e divisão  
✔️ **Botão C (clear)** – Limpa o visor  
✔️ **Cálculo imediato ao clicar "="**  
✔️ **Interface responsiva e amigável**  
✔️ **Design limpo e moderno**
